Overview  

Food Junction is designed to provide an easy and interactive way for users to view restaurant offerings. The website focuses on visual appeal, intuitive navigation, and enhancing user experience.  

-Fully responsive design for desktop and mobile.  
-Modern UI with interactive food cards.  
-Detailed menu with burgers, pizzas, rolls & wraps, starters, and fries.  
-Dynamic hover effects with ingredients, add-ons, ratings, and add-to-cart options.

Data Analysis  

The project includes Excel-based data analysis for menu items, orders, and revenue insights:  
Raw Data Sheet: Menu items, prices, and customer dummy data  
Calculations Sheet: Total revenue, totals per category, conditional analysis  
Dashboard Sheet: Charts for total items sold per category and total revenue per category  
Insights from Excel are embedded into the Word document and presentation slides.
